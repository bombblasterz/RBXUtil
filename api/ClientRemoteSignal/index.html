ClientRemoteSignal

This item only works when running on the client. Client

Client remote signals are used internally by CommunicationSignals to expose serverside remote signals to the client. For e.g, a remote signal on the server can only be representable on the client through a client remote signal binded to it. 

task.spawn(function()
	print(clientRemoteSignal:Wait()) --> "yo"
end)

clientRemoteSignal:Fire("yo")
