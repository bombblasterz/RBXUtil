ClientCommunicationSignal

This item only works when running on the client. Client

A communication signal on the client contains all the methods, members, remote signals and remote properties the server has exposed to it. 

local communicationSignal = CommunicationSignal.GetSignal("ServersideService")

-- Invoke the method "Test" (exposed to the signal):
local method = communicationSignal.Methods.Test
method("Hi")
note

Remote signals and remote properties exposed are created through client remote signals and client remote properties and binded to their server counterparts.
