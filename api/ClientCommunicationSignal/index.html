ClientCommunicationSignal

This item only works when running on the client. Client

A client communication signal contains all the members, remote signals and remote properties binded to it's server counter part. 

-- On the server:
local communicationSignal = CommunicationSignal.new("ServersideService")
communicationSignal:Bind("Test", 123)
communicationSignal:Bind("T", {1, 2, 3})
communicationSignal:Bind("Signal", RemoteSignal.new())

-- On the client:
local communicationSignal = CommunicationSignal.GetSignal("ServersideService")

print(
	communicationSignal.Members.Test,  --> 123
	communicationSignal.Members.T, --> {1, 2, 3}
	communicationRemote.RemoteSignals.Signal 
)
note

Remote signals and remote properties (binded to a signal) are represented differently on the client. They are represented as client remote properties and signals, except they are binded to their serverside counter parts.

For e.g:

-- On the server
local communicationSignal = CommunicationSignal.new("ServersideService")
communicationSignal:Bind("Bo", RemoteSignal.new())

-- On the client
local communicationSignal = CommunicationSignal.GetSignal("ServersideService")
print(ClientRemoteProperty.IsClientRemoteProperty(communicationSignal.RemoteSignals.Bo)) --> true
