A method which yields the current Luau thread until an instance with the name of childName has been found in instance. If no valid instance has been found in under timeout seconds (if specified), the method will return nil.

note

If no valid instance has been found in under 5 seconds, a infinite yield warning will be thrown as a precaution.

tip

If instance has been destroyed, the method will return nil and not leak the thread!
