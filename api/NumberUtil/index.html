NumberUtil

A useful number utility module which provides a lot of useful methods which are commonly used when working with numbers and handling certain edge cases.

For e.g:

local nan = 0/0

print(NumberUtil.IsNaN(nan)) --> true
print(NumberUtil.IsCloseTo(0.1 + 0.2, 0.3)) --> true
