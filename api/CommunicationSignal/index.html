CommunicationSignal

This item only works when running on the server. Server

Communication signals are used to expose members, methods, remote signals and remote properties to the client in a much easier and abstracted fashion. 

-- On the server
local communicationSignal = CommunicationSignal.new("ServersideService")

-- Bind method "Test" to the client:
communicationSignal:BindMethod("Test", function(player, ...)
	warn(("%s said %s"):format(player.Name, tostring(...)))
end))

-- On the client
local communicationSignal = CommunicationSignal.GetSignal("ServersideService")

local method = communicationSignal.Methods.Test
method("Hi")
