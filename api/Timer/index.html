Timer

Timers designed to work like alarms and are used to run code at a specific later time.

For e.g:

local timer = Timer.new(2)
timer.OnTick:Connect(function(deltaTime)
	warn(("Approximately %d seconds have passed"):format(deltaTime))
end)

timer:Start()

Properties#

Boost#

This item is read only and cannot be modified. Read OnlyTimer
</>
Timer.Boost: number

A value which by default is 0. This value is added to the delta time of every approximate frame the timer is updated (or if an update signal was specified, which in case the number fired to the custom update signal). Increasing it will lead to faster timer updates. 

local timer = Timer.new(5)
timer.Boost = 4

timer:Start()

timer.OnTick:Connect(function(deltaTime)
	print(deltaTime) --> 8 (approx) 
end)

Timer.new(
timer:Â number?,
updateSignal:Â RBXScriptSignal?
) â†’Â Timer

Creates and returns a new timer. If timer is not specified, it will default to 0 i.e the timer will be updated every frame. If updateSignal is not specified, it will default to RunService.Heartbeat.

note

If updateSignal is specified, it must be fired with a number as the first argument so that the timer is updated based on that number.

For e.g:

local updateSignal = Instance.new("BindableEvent")
local timer = Timer.new(5, updateSignal.Event)

while true do
	-- The timer will be updated by exactly 5 seconds every approx second:
	updateSignal:Fire(5)
	task.wait(1)
end
