RemoteProperty

This item only works when running on the server. Server

Remote properties are serverside properties which contain getter and setter methods and store a value which can be retrieved by those methods, and are also an abstraction of remote functions. They can also be exposed to the client through communication signals.

tip

Remote properties can also store in player specific values. Making them great for replicating and storing values separate to each player! 

-- Set value 50, but only game.Players.SilentsReplacement should see it.
remoteProperty:SetValue(50, {game.Players.SilentsReplacement})
